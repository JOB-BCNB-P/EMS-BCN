หลักการกรอกข้อมูล
•	ชื่อแท็บ (Tab/Sheet) ต้องสะกดให้ตรงตามคู่มือนี้ทุกตัวอักษร และแถวแรก (แถวที่ 1) ของทุกแท็บคือ "หัวคอลัมน์" ต้องตรงกับชื่อคอลัมน์ในตาราง
•	วันที่ให้กรอกรูปแบบ ปี-เดือน-วัน (ค.ศ.) เช่น 2025-06-27 ระบบจะแปลงเป็น พ.ศ. ให้เองเวลาแสดงผล
•	ปีการศึกษา (academic_year) ให้กรอกเป็น พ.ศ. เช่น 2568
•	ภาคการศึกษา (semester): 1, 2 หรือ 3 (ภาคฤดูร้อน)
•	คอลัมน์ที่ไม่จำเป็นเว้นว่างได้ แต่ห้ามลบหัวคอลัมน์ทิ้ง
•	ตั้งค่าแชร์ Google Sheet เป็น "ทุกคนที่มีลิงก์ = ผู้อ่าน" และเชื่อม Apps Script (โหมดเขียน) เพื่อเพิ่ม/แก้ไขผ่านเว็บได้
•	แท็บที่ระบบสร้าง/จัดการให้อัตโนมัติ (tracking, result_tracking, grade_tracking, file_tracking, doc_request, login_log, permission) เพียงสร้างแท็บและใส่หัวคอลัมน์ไว้ ไม่ต้องกรอกมือ
